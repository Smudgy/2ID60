Gumball Grid, Wallpaper Engine

The script shown here is a snippet of an html5 canvas project called the Gumball Grid. The grid is made up out of shapes that change their appearance and behaviour in accordance to the position of the mouse pointer.

The project is picked from a set of projects made for Wallpaper Engine, an application that allows its users to run webpages as a wallpaper. More projects for this program made by me can be found on Steam or Github.
